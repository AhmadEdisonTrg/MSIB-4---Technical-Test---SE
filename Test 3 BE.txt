Berikut ini adalah contoh implementasi sebuah REST API yang memenuhi ketentuan yang diberikan, menggunakan bahasa pemrograman PHP, dan menggunakan framework Laravel:

Persiapan:
Membuat sebuah proyek baru dengan menjalankan perintah composer create-project --prefer-dist laravel/laravel myapp pada command line.
Membuat konfigurasi database di file .env sesuai dengan server yang digunakan.
Menambahkan package baru yaitu "doctrine/dbal" dan "yajra/laravel-oci8" untuk melakukan ORM
Mengaktifkan migrations dengan perintah php artisan make:migration create_books_table dan membuat migration table yang diperlukan.

Database Migration:
Menjalankan perintah php artisan migrate untuk membuat tabel pada database.

Model:
Membuat model Book, Author, dan Publisher dengan perintah php artisan make:model Book, php artisan make:model Author, php artisan make:model Publisher
Membuat relasi antar model dengan menambahkan method pada masing-masing model. Contoh : pada model Book akan ditambahkan method public function author(){ return $this->belongsTo('App\Author'); } untuk relasi dengan model Author.

CRUD:
Membuat controller dengan perintah php artisan make:controller BookController dan menambahkan method yang diperlukan untuk CRUD.
Menambahkan route pada web.php untuk setiap method pada controller yang dibuat.

Authentication:
Menambahkan package "tymon/jwt-auth" dengan perintah composer require tymon/jwt-auth
Menjalankan perintah php artisan jwt:secret untuk mengenerate secret key
Membuat guard baru pada config/auth.php
Membuat middleware baru dengan perintah php artisan make:middleware JwtAuthMiddleware dan menambahkan pada route yang diperlukan.

Validator:
Menambahkan validator pada setiap method CRUD yang dibuat dengan menggunakan request validation pada laravel.
Unit test (optional):
Membuat test case dengan perintah php artisan make:test BookTest
Menambahkan test method pada test case yang dibuat
Menjalankan test dengan perintah phpunit
